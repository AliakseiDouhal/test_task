I used a starter app which contain many additional dependences, please do not pay attention to them
## How to run?

Install packages.
```shell
$ yarn
```

Run Android Project.
```shell
$ react-native run-android
```

Run iOS Project.
```shell
$ cd ios && pod install
$ cd .. && react-native run-ios
```
